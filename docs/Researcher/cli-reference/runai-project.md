## Description

Set a default Project.

## Synopsis

``` shell

runai project set <project-name>
    [--loglevel value] 
    [--help | -h]

```
## Options

<project-name\> the name of the Project you want to set as default. Mandatory.


### Global Flags

--loglevel (string)

> Set the logging level. One of: debug | info | warn | error (default "info").


--help | -h

>  Show help text.

## Output

None
